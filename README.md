# 🛒 Shopy Stream: AI-Powered Shoppable Streaming Platform

A modern, interactive streaming platform where viewers can click on objects inside a playing video to instantly discover and purchase them. Powered by cutting-edge AI object detection and seamlessly integrated with live e-commerce stores.

---

## ✨ Features
- **AI Object Detection:** Automatically detects objects (clothes, electronics, furniture, etc.) in video streams using **YOLOv8-World** open-vocabulary detection and **SAM3** (Segment Anything).
- **Interactive Video Player:** Viewers can click on highlighted glowing cyan bounding boxes during video playback to see product details.
- **E-Commerce Integration:** Instantly maps detected objects to real products in your e-commerce store or falls back to Amazon affiliate links.
- **Scalable Backend:** Robust **Spring Boot** architecture using MongoDB for storing time-series metadata of objects in the video.
- **Dockerized ML Service:** A standalone Python microservice handles the heavy lifting of running frame-by-frame analysis.
- **Automated Cloud Deployment:** Fully configured CI/CD pipeline using **GitHub Actions** and Docker Compose for zero-downtime AWS EC2 deployments.

## 🏗️ Getting Started (Local Development)

### Prerequisites
- Node.js (v18+)
- Java 17+
- Python 3.10+
- Docker & Docker Compose

### Start the Entire Stack (One Command)
Because we have fully containerized the application, you can start the Frontend, Backend, ML Service, MongoDB, and Postgres with a single command:
```bash
docker-compose up -d --build
```
*The web app will instantly be available at http://localhost:80*

## ☁️ Cloud Deployment (AWS)
This project is configured for automated cloud deployment!
1. Launch an Ubuntu EC2 instance on AWS and paste the contents of `deploy-setup.sh` into the **User Data** field to automatically install Docker.
2. Add your EC2 Public IP and SSH Key to your GitHub Repository Secrets (`EC2_HOST`, `EC2_SSH_KEY`).
3. Push your code to the `main` branch. GitHub Actions will automatically deploy the newest version of Shopy Stream to your server!

## 🎥 How it Works Under the Hood
1. **Upload:** A video is uploaded or selected.
2. **Analysis:** The Spring Boot backend forwards the video to the Python ML Service.
3. **Detection:** YOLOv8-World analyzes the video, generating bounding boxes and classifying objects (e.g., "laptop", "tie"). SAM3 provides complex segmentation tracking.
4. **Metadata Storage:** The detections are enriched with product metadata and stored chronologically in MongoDB.
5. **Playback:** As the user watches the video, the React player dynamically syncs with the timeline metadata (using `useEffect` and `onTimeUpdate`) to draw pixel-perfect interactive overlays on the playing video.
